Vim Snake

Vim Snake is a quick and fun 1 day sprint project to explore HTML5 canvas. The game is built with canvas/javascript/jquery. I used sinatra for the backend and twitter for the login and leaderboard.

At the time of development, I was learning Vim and decided to use the same navigation controls for snake. The snake can only eat food in INSERT mode, and you cannot change direction until escaping INSERT mode

On August 16, Vim Snake made the front page of Hacker News which resulted in significant exposure and positive feedback.

Box Report

Ever wondered what's taking up all that space in your Dropbox? Visualize and explore the contents of your Dropbox to easily find the files or folders taking up the most space.

I was responsible for the entire project. Site built with Sinatra. Authentication using Dropbox API. Graphs with Google Charts

The most interesting part of the project was getting the size of every folder, which involved recursively drilling into each folder and its contents to add up the total size.
